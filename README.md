## Descrição do Projeto:

O projeto realizado faz parceria com a instituição educacional FMABC, Faculdade de Medicina do ABC. O software a ser desenvolvido será voltado ao ensino de 
Histologia com apoio de recursos digitais, tendo fins didáticos e científicos. O sistema contará com imagens ilustrativas de estruturas citológicas, sendo possível 
ampliá-las e anotar matérias ao lado delas.

## Implementação do Código:
1. Copie o link que está dentro deste repositório na aba "Code", clicando no botão verde escrito "Code";
2. Crie uma pasta na sua área de trabalho, abra o cmd (Command Prompt) dentro desta e digite `git clone <link_github>`;
3. Digite `dir` se preferir saber as configurações e organizações dos códigos;
4. Digite `code .` para abrir a pasta no Visual Studio Code;
5. No terminal Git Bash, de preferência, digite `cd backendJavaScript` e depois `npm start` para rodar o backend;
6. Abra um outro terminal Git Bash e digite `cd atlas_digital` e `flutter pub get` para instalar as dependências;
7. Por último, digite nesse segundo terminal `flutter run -d chrome` para simular o sistema em um servidor local.
